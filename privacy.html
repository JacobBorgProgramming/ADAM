...But what data do we collect?

  • The day of the week (As in: Monday, Tuesday, ...)

How is this data used?

This data is used in order to monitor page level usage and improve ADAM.

Huh?

We use Counter to track usage of ADAM.

By using Counter and not other analytical services, we are able to track only what is needed in order to provide a better product.
