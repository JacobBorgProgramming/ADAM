About

What is A.D.A.M.?

A.D.A.M. stands for "Algorithmic Discourse And Memeing".

A.D.A.M. is a program that generates memes based on an index of current events, trends, and emotions.

Will there ever be a prompt feature?

Maybe. Right now development is focussed on the primary algorithm that does not take prompts, but I may write an algorithm that does work with prompts.

Is there bias?

Of course there is.

Any algorithm or artificial intelligence will be biased. It is impossible to keep one's own bias out of a program like this. If any developer tells you otherwise, they are being dishonest or are naive.
